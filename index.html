Web Development

Monashee Mountain Multimedia builds custom software.

Derek specializes in Magento Enterprise and WordPress plugin and theme development. Derek has built and maintained other CMS frameworks along with custom web applications for enterprise clients.

MMM also creates custom digital signage solutions such as digital cinema pre-shows and custom digital advertising solutions.

Derek has been coding on computers dating back to when he attended the University of Manitoba from 1987 to 1991. He wrote his first code in 1982 on a Commodore VIC 20. Back then, after school he would run to the store to buy the latest Compute magazine which had video games code in it and he would input and tweak the code to build video games.

His personal website is at Derek.Marcinyshyn.com.

Web Hosting

Revelstoke Web Hosting is a custom built server for high traffic sites and web site owners that want fast page load times. The servers are located on the Peer 1 network based in Vancouver, BC. RWH uses the LiteSpeed web server which is up to 6x faster than Apache web server. All accounts use Varnish reverse proxy engine to dramatically speed up your page loads.

We generally serve static site assets from a content delivery network such as Amazon CloudFront. This site uses Net DNA for the Twitter Bootstrap assets and Amazon CloudFront for the images.
